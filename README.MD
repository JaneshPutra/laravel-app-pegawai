## ğŸ› ï¸ Instalasi
1. Clone repository:
   ```bash
   git clone https://github.com/JaneshPutra/laravel-app-pegawai.git
   ```
2. Masuk ke folder project:
   ```bash
   cd laravel-hr-attendance
   ```
3. Install dependencies:
   ```bash
   composer install
   npm install && npm run dev
   ```
4. Copy `.env.example` ke `.env` dan sesuaikan konfigurasi database.
5. Jalankan migrasi:
   ```bash
   php artisan migrate --seed
   ```
6. Jalankan server:
   ```bash
   php artisan serve
   ```
